# Earn with DG

## How to build APK from GitHub Actions

1. Open GitHub repository
2. Go to Actions
3. Select **Android Debug APK**
4. Click **Run workflow**
5. Wait for build to finish
6. Open latest run
7. Download artifact: `earn-with-dg-debug-apk`
8. Extract ZIP
9. Install `app-debug.apk` on Android phone
